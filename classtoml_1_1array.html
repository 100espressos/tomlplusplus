toml::array class final

A TOML array.

Contents

  • Reference
    • Base classes
    • Public functions

The interface of this type is modeled after std::vector so things mostly work as you'd expect them to with a vector:

auto table = toml::parse("arr = [1, 2, 3, 4, 'five']"sv);

auto& arr = *table.get_as<toml::array>("arr");

for (size_t i = 0; i < arr.size(); i++)
{
    arr[i].visit([=](auto&& el) noexcept
    {
        std::cout << el << ", ";
    });
}

// prints: 1, 2, 3, 4, "five"
